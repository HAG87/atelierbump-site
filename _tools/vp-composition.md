![vp?guides](/assets/images/vp-guides.png)

## Features

* 4-field grid (2x2)
* 9-field grid (3x3)
* Position a guide in the viewport with the mouse. Possible positions: vertical, horizontal, or both.
* You can select the color of the guides.
