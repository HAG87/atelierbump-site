Spatial Reference guides

Refernece tools are intended to improve and speed up architecture modeling, or any case where the use of dimensin references are needed. It allows to create construction guides, for accurately position and model objects. This guides can act as rulers, displaying custom measurement units in the viewport. The main functionality of this toolset consist in the creation of two types of reference objects: linear guides and protractors.
